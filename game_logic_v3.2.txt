БЛОК A — СВІТ І СТРУКТУРА

────────────────────────────────────────────────────────────────
A1. СТРУКТУРА СВІТУ І СЕРВЕР
────────────────────────────────────────────────────────────────

СЕРВЕР:
  - Максимум 8 гравців (MaxPlayers = 8)
  - Океан і зони — спільні для всіх 8 гравців
  - Ресурси на островах — спільні (хто перший підібрав)
  - Рюкзак, музей, льох, склад — особисті для кожного гравця
  - Аукціон — глобальний між УСІМА серверами

ХАБ (Острів гравця) — стартова локація
  └── NPC Туторіал         → пояснює механіки новому гравцю
  └── NPC Магазин          → продаж риби за фіксованими цінами
  └── Музей колекцій       → пасивний дохід, колекція назавжди
  └── Льодовий льох        → зберігання риби без псування
  └── Склад ресурсів       → окрема будівля, зберігання ресурсів
  └── Верстак              → [заблоковано в MVP — Хвиля 2]
                             При взаємодії: "Буде доступно пізніше"
  └── Дошка аукціону       → глобальна торгівля між гравцями
  └── Дошка квестів        → 3 активних квести (деталі — Хвиля 2)
  └── Причал               → вихід в океан на особистому човні
                             або паті-кораблі

ОКЕАН
  └── Зона 1: Мілководдя / Shallows  → безкоштовна
      └── 3x FishingSpot у воді (радіус 80 studs кожен)
      └── Острів 1 — лише збір ресурсів
          └── NPC-торговець Острова 1 (ресурси Зони 1)
  └── Зона 2: Рифи / The Reefs       → 15 000 монет/тиждень
      └── 3x FishingSpot у воді (радіус 80 studs кожен)
      └── Острів 2 — лише збір ресурсів
          └── NPC-торговець Острова 2 (ресурси Зони 2)
      └── Буйки — візуальна межа зони
      └── Поліцейський катер біля буйків (перша купівля ліцензії)
  [Зони 3-4 — Хвиля 2]

ПРИМІТКИ:
  ! NPC магазин ресурсів на хабі ПРИБРАНО. Ресурси купуються
    виключно у NPC-торговців на островах або через аукціон.
  ! Якщо гравець стрибає у воду — вмирає миттєво і
    респавниться на хабі. Предмети не втрачаються.
    Механіка існує як обмежувач — переміщення лише човном.


────────────────────────────────────────────────────────────────
A2. ЧОВЕН І ПЕРЕМІЩЕННЯ
────────────────────────────────────────────────────────────────

ПРИНЦИП:
  - Гравець керує човном в реальному часі
  - Керування: WASD / віртуальний джойстик / лівий стік
    (кросплатформа — ПК, мобільний, консоль автоматично)
  - Без інерції в MVP
  - Між зонами — loading screen (оптимізація для слабких ПК)
  - Рибалка виключно з човна у відкритій воді
  - Острови — виключно для збору ресурсів і взаємодії з NPC
  - Кожен особистий човен доступний лише його власнику
  - Позиція човна і гравця зберігається при виході з гри
    (через DataStore) — при поверненні гравець там де вийшов

БАЗОВИЙ ЧОВЕН (MVP):
  - Швидкість: 1x
  - Вантажність: не обмежена в MVP
  - Візуал: стандартний дерев'яний човен

АПГРЕЙДИ ЧОВНА (Хвиля 2):
  Рівень 2 → швидкість 1.3x, вантаж 15, новий візуал
  Рівень 3 → швидкість 1.6x, вантаж 20, новий візуал
  Рівень 4 → швидкість 2.0x, вантаж 30, унікальний візуал
  Вартість: монети + острівні ресурси


────────────────────────────────────────────────────────────────
A3. ЗОНИ І ЛІЦЕНЗІЇ
────────────────────────────────────────────────────────────────

  Зона 1 (Мілководдя)    → безкоштовна, завжди доступна
  Зона 2 (Рифи)          → 15 000 монет/тиждень
  Зона 3 (Відкрите море) → ціна TBD               [Хвиля 2]
  Зона 4 (Глибини)       → ціна TBD               [Хвиля 2]

  Ліцензія діє тиждень і списується автоматично при продовженні.
  Без ліцензії — вхід в зону заблокований буйками.
  Ліцензії — регулярний дренаж економіки.

МЕЖА ЗОН:
  - Буйки — візуальна межа між зонами
  - Поліцейський катер NPC біля буйків Зони 2
  - При підпливанні без ліцензії → діалог з катером
    з пропозицією купити ліцензію
  - Перша купівля — у катера на місці
  - Подальше управління і автооплата — в меню підписок на хабі

МЕНЮ ПІДПИСОК:
  - Окреме меню у гравця з переліком усіх зон
  - Для кожної зони окремо вмикається/вимикається автооплата
  - Автооплата розблоковується після першої купівлі ліцензії
  - Якщо автооплата вимкнена і ліцензія закінчилась —
    зона блокується до ручної оплати через меню або катера
  - Таймер до закінчення ліцензії видно в меню


────────────────────────────────────────────────────────────────
A4. ПАТІ-СИСТЕМА
────────────────────────────────────────────────────────────────

ПАТІ-КОРАБЕЛЬ:
  - При створенні паті на причалі хабу спавниться окремий
    паті-корабель (не особистий човен жодного з гравців)
  - Вигляд і характеристики визначаються найнижчим рівнем
    особистого човна серед членів паті
  - Паті-корабель не апгрейдиться окремо
  - В паті-кораблі можуть перебувати всі члени паті (до 4)
  - Особисті човни залишаються незмінними

БОНУСНІ СЛОТИ В ПАТІ:
  Формула: рівень паті-корабля = кількість бонусних слотів
  за кожного іншого члена паті (окрім себе)

  ┌─────────────────┬─────────────────────────────────────────┐
  │ Рівень корабля  │ Бонус слотів за кожного члена паті      │
  ├─────────────────┼─────────────────────────────────────────┤
  │ Рівень 1 (MVP)  │ +1 слот                                 │
  │ Рівень 2        │ +2 слоти                                │
  │ Рівень 3        │ +3 слоти                                │
  │ Рівень 4        │ +5 слотів                               │
  └─────────────────┴─────────────────────────────────────────┘

  Приклад MVP: паті-корабель 1 рівня + повне паті (4 гравці)
  → кожен отримує +3 бонусних слоти рюкзака

  Бонус активний поки паті існує (і на хабі, і в океані)

ФАНТОМНІ СЛОТИ:
  При розпаді паті якщо предметів більше ніж реальних слотів:
  - Надлишкові предмети переходять у фантомні слоти
  - Фантомний слот зникає коли предмет переміщено у сховище
    або займає місце звільненого реального слоту
  - Риба у фантомних слотах псується як звичайно
  - Ресурси у фантомних слотах не псуються

  Годинник і фаза дня — в інтерфейсі гравця

  │ Фаза     │ Тривалість│ Вплив на рибалку                   │
  ├──────────┼───────────┼────────────────────────────────────┤

────────────────────────────────────────────────────────────────
B2. СИСТЕМА ПОГОДИ
────────────────────────────────────────────────────────────────

  Погода змінюється кожні 8-12 хвилин.
  При кожній зміні — рандомний вибір за таблицею шансів.
  Та сама погода може повторитись.
  Іконка погоди і таймер до зміни — в інтерфейсі гравця.
  Всі модифікатори — мультиплікативні.

  ┌───────────┬───────┬─────────────────────────────────────────┐
  │ Погода    │ Шанс  │ Ефект на рибалку                        │
  ├───────────┼───────┼─────────────────────────────────────────┤
  │ Ясно      │  35%  │ x1.0 стандарт, швидке клювання          │
  │ Хмарно    │  30%  │ x1.1 до Rare, середня швидкість         │
  │ Дощ       │  20%  │ x1.2 до Rare/Epic, повільне клювання    │
  │ Туман     │  10%  │ x0.7 Common / x1.2 Uncommon /           │
  │           │       │ x1.4 Rare / x1.3 Epic / x1.2 Legendary  │
  │           │       │ + ексклюзиви Зони 1: Димчастий лин      │
  │           │       │ + ексклюзиви Зони 2: Молочна химера     │
  │           │       │ + тимчасові FishingSpot у відкритій воді│
  │ Гроза     │   5%  │ x1.35 до Epic/Legendary, дуже повільно  │
  │           │       │ + ексклюзиви Зони 1: Громова щелепа     │
  │           │       │ + ексклюзиви Зони 2: Блискавична барак. │
  │           │       │ + тимчасові FishingSpot у відкритій воді│
  └───────────┴───────┴─────────────────────────────────────────┘

  БОНУС МУЗЕЮ ЗА ПОГОДУ ЛОВУ: ВІДХИЛЕНО — риба коштує і приносить
  дохід однаково незалежно від погоди, під час якої її спіймано


────────────────────────────────────────────────────────────────
B3. МЕХАНІКА РИБАЛКИ І МІНІГРА
────────────────────────────────────────────────────────────────

FISHINGSPOT:
  - 3 FishingSpot у кожній зоні, радіус 80 studs кожен
  - Детектування входу: візуальний ефект на воді +
    текстове повідомлення на екрані гравця
  - У FishingSpot → множник x1.2 до всіх шансів рідкості
  - Поза FishingSpot → множник x0.8 до всіх шансів рідкості
  - Під час Грози/Туману — тимчасові FishingSpot у відкритій воді

ПОВНИЙ ЦИКЛ ЛОВЛІ:
  1. Гравець підпливає човном до FishingSpot (або рибалить
     у відкритій воді з множником x0.8)
  2. Натискає "Закинути вудку"
  3. Поплавок анімується — гравець чекає (таймер по вудці)
  4. Поплавок смикається → з'являється кнопка "Тягни!"
  5. Гравець натискає → відкривається таймінг-бар
  6. Є 3-5 секунд щоб натиснути в потрібній зоні
  7. Влучив → система визначає рибу за формулою
  8. Не натиснув → риба зірвалась → крок 2 (без штрафу)

ТАЙМЕР ОЧІКУВАННЯ ПОПЛАВКА:
  Рівень 1 → 8-12 секунд
  Рівень 2 → 6-9 секунд
  Рівень 3 → 5-7 секунд
  Рівень 4 → 3-5 секунд
  Рівень 5 → 2-4 секунди
  Рівень 6 → 1-3 секунди

МІНІГРА — ТАЙМІНГ-БАР:
  Повзунок рухається туди-сюди.
  Гравець натискає один раз — результат за зоною натискання.

  ┌──────────────────────────────────────────────────────┐
  │ 🔴 Слабка │ 🟡 Середня │ 🟢 Хороша │ 🔵 Ідеальна  │
  │   37%     │    35%     │    20%    │      8%      │
  │   x0.5    │    x1.0    │    x1.5   │     x2.5    │
  └──────────────────────────────────────────────────────┘

  ВПЛИВ РІВНЯ ВУДКИ НА БАР:
  Рівень 1 → повільний повзунок,     синя зона  8%
  Рівень 2 → середній повзунок,      синя зона 10%
  Рівень 3 → швидший повзунок,       синя зона 13%
  Рівень 4 → швидкий повзунок,       синя зона 16%
  Рівень 5 → дуже швидкий повзунок,  синя зона 19%
  Рівень 6 → максимальний повзунок,  синя зона 23%

  Вища вудка = швидший повзунок АЛЕ більша синя зона.
  Механічна прокачка вудки завершена на Рівні 6.
  Рівні 7+ — лише косметичні скіни (Хвиля 3).

СКІНИ ВУДКИ (Хвиля 3):
  - Косметичні → лише візуал, без впливу на механіку
  - Ефектні → візуал + унікальний пасивний бонус
    (наприклад: +шанс рідкості, +шанс префіксу тощо)
  - Деталі ефектів — Хвиля 3

ФОРМУЛА ВИЗНАЧЕННЯ РИБИ:
  Фінальний шанс =
    базовий шанс рідкості
    × множник зони бару (x0.5 / x1.0 / x1.5 / x2.5)
    × модифікатор погоди
    × модифікатор часу доби
    × множник FishingSpot (x1.2 у споті / x0.8 поза спотом)

  Потім перевіряється чи активні погодні/нічні ексклюзиви.
  Потім перевіряється шанс префіксу.

БАЗОВІ ШАНСИ РІДКОСТІ:
  Common     → 55%
  Uncommon   → 25%
  Rare       → 12%
  Epic       →  6%
  Legendary  →  2%

БЛОК F — ПРОГРЕСІЯ І ЗБЕРЕЖЕННЯ
================================================================

────────────────────────────────────────────────────────────────
F1. ТУТОРІАЛ
────────────────────────────────────────────────────────────────

  КРОК 1  → NPC вітає, пояснює ціль гри
  КРОК 2  → NPC видає вудку Рівня 1 безкоштовно
  КРОК 3  → NPC пояснює рюкзак, псування і автоскладання
  КРОК 4  → Стрілка веде до причалу → гравець виходить в океан
  КРОК 5  → Гравець підпливає до FishingSpot і ловить рибу
             (перша риба гарантовано Common, псування не діє)
  КРОК 6  → Гравець повертається в хаб і продає рибу NPC-магазину
  КРОК 7  → NPC показує музей і пасивний дохід
  КРОК 8  → NPC показує льох, склад і аукціон
  КРОК 9  → NPC пояснює острови, NPC-торговців, зони і буйки
  КРОК 10 → Туторіал завершено — гравець вільний

  Автоматично пропускається якщо гравець вже грав.


────────────────────────────────────────────────────────────────
F2. СТАРТОВИЙ СТАН ГРАВЦЯ
────────────────────────────────────────────────────────────────

  При першому вході (DataStore порожній):
  - Монети:              1000
  - Вудка:               Рівень 1 (видається туторіалом)
  - Рюкзак:              Рівень 1 (10 слотів риби / 10 ресурсів)
  - Льодовий льох:       Рівень 1 (5 слотів)
  - Склад ресурсів:      Рівень 1 (100 одиниць)
  - Прапорець туторіалу: false
  - Ліцензії:            лише Зона 1 (безкоштовна, завжди)
  - Позиція:             хаб (SpawnLocation)


────────────────────────────────────────────────────────────────
F3. ЗБЕРЕЖЕННЯ ДАНИХ (DataStore)
────────────────────────────────────────────────────────────────

ЩО ЗБЕРІГАЄТЬСЯ:
  - Кількість монет
  - Рюкзак: { name, prefix, rarity, spoilTimer } (риба)
             { name, count } (ресурси)
  - Склад ресурсів: { name, count } + рівень складу
  - Музей: { name, prefix, rarity, count } по зонах
  - Льодовий льох: { name, prefix, rarity } + рівень льоху
  - Час останнього виходу (Unix timestamp)
  - Позиція човна і гравця (Vector3)
  - Рівень вудки
  - Рівень рюкзака
  - Активні ліцензії: { zone, expiresAt, autoRenew }
  - Активні та неактивні лоти на аукціоні
  - Прапорець туторіалу

  ПРИМІТКА: бонус за друга онлайн НЕ зберігається —
  визначається в реальному часі через Roblox Friends API.

ЛОГІКА ПРИ ВХОДІ — МУЗЕЙ (пасивний дохід):
  1. Зчитати час виходу
  2. Хвилини офлайн (max 2880)
  3. Для кожного екземпляра в музеї: хвилини × дохід/хв
     (з урахуванням бонусів префіксу і колекції)
  4. Нарахувати монети
  5. Показати повідомлення "Поки тебе не було, музей заробив X монет!"

ЛОГІКА ПРИ ВХОДІ — ПСУВАННЯ:
  1. Зчитати рюкзак і час виходу
  2. Хвилини офлайн
  3. Для кожної риби в рюкзаку: spoilTimer - хвилини офлайн
  4. ≤ 0 → видалити, > 0 → оновити таймер
  5. Показати повідомлення якщо щось зіпсувалось

ЛОГІКА ПРИ ВХОДІ — ЛІЦЕНЗІЇ:
  1. Перевірити expiresAt кожної ліцензії
  2. autoRenew = true і монет достатньо → списати, продовжити
  3. autoRenew = true але монет недостатньо → анулювати,
     повідомити гравця
  4. autoRenew = false і термін вийшов → анулювати,
     повідомити гравця
  5. Показати стан кожної ліцензії в меню підписок

ВИЗНАЧЕННЯ ДРУЗІВ:

  Перевірка при вході і при зміні складу сервера.
  Не зберігається в DataStore — визначається в реальному часі.

БОНУС ЗА ДРУГА ОНЛАЙН:
  - +10% до пасивного доходу музею за кожного друга
    що перебуває на тому ж сервері в цей момент
  - Максимум +30% (3 друзі одночасно)
  - Бонус симетричний — кожен гравець у групі отримує +30%
    якщо з ним 3 друзі на сервері
  - Повне паті (4 гравці) = всі отримують +30%
  - Бонус зникає як тільки друг виходить або переходить
    на інший сервер

БОНУС ЗА ЗАПРОШЕННЯ:
  - Якщо друг реєструється за запрошенням →
    обидва отримують по 100 монет одноразово
